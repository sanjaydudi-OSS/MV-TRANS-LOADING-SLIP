🌐 GitHub Pages — Get a Permanent Public Link

GitHub Pages gives you a free permanent URL like yourusername.github.io/mv-trans that you can bookmark and share with anyone. It takes about 5 minutes to set up.

1
Create a free GitHub account
Go to github.com and sign up for a free account. Choose a username like mvtrans or your own name.
2
Create a new repository
Click the + icon → New repository. Name it mv-trans (or anything you like). Set it to Public. Check "Add a README file". Click Create repository.
3
Download this file and rename it
Download this HTML file. Rename it exactly to index.html.
4
Upload the file to your repository
In your repository on GitHub, click "Add file" → "Upload files". Drag and drop your index.html file. Add a commit message like Add memo system. Click Commit changes.
5
Enable GitHub Pages
Go to your repository Settings (top tab). Scroll down to Pages in the left sidebar. Under Branch, select main and folder / (root). Click Save.
6
Get your permanent URL
After ~1 minute, refresh the Pages settings. You'll see your URL: https://yourusername.github.io/mv-trans/ — share this with your team!
Updating later: Whenever you want to update the app, just re-upload the new index.html file to the same repository. GitHub Pages updates automatically within 1–2 minutes.
